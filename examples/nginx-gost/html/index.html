Hello from docker-gost!

This page is served by nginx compiled against OpenSSL with the gost-engine submodule. The connection between the proxy and this server is secured with a self-signed GOST 2012 certificate.

Inspect the certificate

openssl s_client \
  -connect localhost:443 \
  -cipher GOST2012-KUZNYECHIK-KUZNYECHIKOMAC \
  </dev/null

Verify available GOST ciphers

openssl ciphers | tr ':' '\n' | grep GOST
